Privacy Policy

Effective date: 2026-03-17

This application is used to publish creator-approved content to TikTok via official TikTok APIs.

Data We Collect

  • TikTok OAuth data (open_id, access token, refresh token)
  • Posting metadata (caption, hashtags, publish status, timestamps)

How We Use Data

  • Authenticate the creator account
  • Upload and publish content only after explicit user approval
  • Track posting status and reliability logs

Data Sharing

We do not sell personal data. Data is only used for TikTok publishing workflow.

Data Retention

Tokens and logs are retained only as needed for operation and security.
